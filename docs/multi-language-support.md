# Поддержка других языков

> Заметки для будущей реализации мультиязычности

## Текущее состояние

- Пара языков: **English ↔ Russian**
- Таблица `words` хранит английские слова
- Таблица `word_meanings` хранит русские переводы
- Лемматизация: `wink-lemmatizer` (только английский)

## Yandex Dictionary API

Поддерживаемые пары:
- `en-ru`, `ru-en` (текущие)
- `en-de`, `de-en` (английский ↔ немецкий)
- `en-es`, `es-en` (английский ↔ испанский)
- `de-ru`, `ru-de` (немецкий ↔ русский)
- `es-ru`, `ru-es` (испанский ↔ русский)
- `en-fr`, `fr-en`, `fr-ru`, `ru-fr` (французский)
- И другие

## Варианты архитектуры

### Вариант 1: Расширение текущей схемы (рекомендуется)

Минимальные изменения, текущая схема уже готова:

```
words
├── id, text, language ('en', 'de', 'es'), lemma, transcription
└── word_meanings
    ├── translation, translationLanguage ('ru', 'en', ...)
    └── partOfSpeech, frequency, examples...
```

**Что нужно сделать:**
1. Расширить `detectLang()` для новых языков
2. Добавить лемматизаторы по мере необходимости
3. UI: выбор языковой пары в настройках

**Плюсы:** простота, обратная совместимость
**Минусы:** лемматизация для каждого языка отдельно

### Вариант 2: Универсальные пары (many-to-many)

```
words
├── id, text, language
└── translations (many-to-many через pivot table)
    ├── source_word_id, target_word_id
    └── context, pos, frequency
```

**Плюсы:** любые комбинации языков, переиспользование слов
**Минусы:** сложнее, требует рефакторинга

## Детекция языка

Текущая реализация (`dictionary-service.ts:91`):
```ts
const CYRILLIC_RE = /[а-яёА-ЯЁ]/;
function detectLang(text: string): string {
  return CYRILLIC_RE.test(text) ? 'ru-en' : 'en-ru';
}
```

Расширенная версия:
```ts
const CYRILLIC_RE = /[а-яёА-ЯЁ]/;
const GERMAN_RE = /[äöüßÄÖÜ]/;
const SPANISH_RE = /[áéíóúñüÁÉÍÓÚÑÜ¿¡]/;

function detectLang(text: string, targetLang = 'ru'): string {
  if (CYRILLIC_RE.test(text)) return `ru-${targetLang === 'ru' ? 'en' : targetLang}`;
  if (GERMAN_RE.test(text)) return `de-${targetLang}`;
  if (SPANISH_RE.test(text)) return `es-${targetLang}`;
  return `en-${targetLang}`;
}
```
